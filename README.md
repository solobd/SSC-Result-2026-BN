# SSC Result 2026 – Bantra Secondary School

একটি স্ট্যাটিক ওয়েব অ্যাপ্লিকেশন, যা দিয়ে শিক্ষার্থীরা তাদের Roll ও Registration নম্বর ব্যবহার করে SSC পরীক্ষার ফলাফল দেখতে পারে। ফ্রন্টএন্ড সম্পূর্ণভাবে HTML/CSS/JS দিয়ে তৈরি এবং JSON ফাইল থেকে ডেটা লোড করে।

## বৈশিষ্ট্য

- Roll/Registration নম্বর দিয়ে রেজাল্ট সার্চ
- শিক্ষার্থীর প্রোফাইল ছবি প্রদর্শন
- রেসপন্সিভ ডিজাইন (মোবাইল-ফ্রেন্ডলি)
- SweetAlert2 দিয়ে ইউজার-ফ্রেন্ডলি নোটিফিকেশন/অ্যালার্ট
- Apache `.htaccess` কনফিগারেশন সহ

## প্রজেক্ট স্ট্রাকচার

```
.
├── index.html          # মূল অ্যাপ্লিকেশন (UI + লজিক)
├── .htaccess            # Apache সার্ভার কনফিগারেশন
├── assets/
│   ├── jb_logo.png       # স্কুল/প্রতিষ্ঠানের লোগো
│   └── default.png       # ডিফল্ট প্রোফাইল ছবি
├── data/
│   ├── students.json     # শিক্ষার্থীদের তথ্য (নাম, রোল, রেজিস্ট্রেশন ইত্যাদি)
│   └── result.json        # রোল অনুযায়ী রেজাল্ট স্ট্যাটাস
└── students/             # শিক্ষার্থীদের প্রোফাইল ছবি (roll.jpg ফরম্যাটে)
```

## ব্যবহারের নিয়ম

1. যেকোনো Apache/Nginx হোস্টিং বা GitHub Pages-এ ফাইলগুলো আপলোড করুন।
2. `index.html` ব্রাউজারে ওপেন করুন অথবা ডোমেইনে ভিজিট করুন।
3. Roll ও Registration নম্বর দিয়ে সার্চ করে রেজাল্ট দেখুন।

## ডেটা আপডেট

- নতুন শিক্ষার্থী যোগ করতে `data/students.json`-এ এন্ট্রি যোগ করুন এবং `students/` ফোল্ডারে সংশ্লিষ্ট roll নম্বর অনুযায়ী ছবি রাখুন।
- রেজাল্ট প্রকাশ করতে `data/result.json`-এ সংশ্লিষ্ট roll-এর মান আপডেট করুন।

## ⚠️ গুরুত্বপূর্ণ নোট (প্রাইভেসি)

এই রিপোজিটরিতে **প্রকৃত শিক্ষার্থীদের ব্যক্তিগত তথ্য** (নাম, পিতা-মাতার নাম, রোল, রেজিস্ট্রেশন নম্বর, ছবি) রয়েছে। এই প্রজেক্ট **Public GitHub রিপোজিটরিতে পুশ করা নিরাপদ নয়**। সুপারিশ:

- রিপোজিটরি **Private** রাখুন, অথবা
- `data/` ও `students/` ফোল্ডার `.gitignore`-এ যোগ করে শুধু কোড পুশ করুন এবং ডেটা আলাদাভাবে (নিরাপদ চ্যানেলে) হোস্ট করুন।

## লাইসেন্স

এই প্রজেক্ট Bantra Secondary School-এর ব্যবহারের জন্য তৈরি। বিতরণ বা পুনঃব্যবহারের আগে প্রতিষ্ঠানের অনুমতি নিন।
